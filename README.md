# 🏥 Medical Insurance Premium Prediction Using Machine Learning

## 📖 Overview
This project focuses on predicting medical insurance costs based on individual customer attributes. The notebook covers the complete data science pipeline, from exploratory data analysis and visualization to training, evaluating, and comparing various machine learning regression models, including Linear Regression, Decision Trees, Random Forests, and Artificial Neural Networks (ANN).

## 🎯 Objective
The primary objective is to build and compare multiple regression models to accurately forecast medical insurance premiums. By analyzing demographic and health-related data, the project aims to identify the best-performing algorithm for financial prediction in the healthcare insurance domain.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Machine Learning:** Scikit-Learn (Linear Regression, Decision Tree, Random Forest)
* **Deep Learning:** TensorFlow / Keras (ANN)
* **Data Manipulation & Visualization:** pandas, NumPy, Matplotlib, Seaborn

## 🧠 Key Learnings & Skills Demonstrated
* **Problem Formulation:** Understanding the business case for regression tasks in the insurance industry.
* **Exploratory Data Analysis (EDA):** Importing datasets and investigating feature distributions and missing values.
* **Data Visualization:** Creating plots to uncover correlations between patient attributes (like age, BMI, smoking status) and insurance charges.
* **Data Preparation:** Creating structured training and testing datasets.
* **Model Training & Comparison:** Implementing and evaluating traditional machine learning algorithms alongside a deep learning Neural Network to determine the most accurate predictive model.
* **Performance Evaluation:** Assessing trained model performance using standard regression metrics.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Medical-Insurance-Premium-Prediction-Using-Machine-Learning.git

2. Navigate into the downloaded project folder:
   cd Medical-Insurance-Premium-Prediction-Using-Machine-Learning

3. Install the required dependencies:
   pip install pandas numpy scikit-learn tensorflow matplotlib seaborn jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to run the analysis and train the models.
